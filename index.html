Design focus points for 2017 - 1/2/2017

Keeping up with design trends is essential to web development. More and more statistics show that an up-to-date responsive modern website leads to more searches, higher retention rate, and increased traffic.

In 2017 i'll focus on these few points: Accessibility, Responsiveness, User experience / User flow, Page speed, and Security

Accessibility means making the web page accessible for everyone, whether they're on a tablet or can't afford the latest iphone7. I'll focus on simpler yet intuitive web elements while following the WCAG guidelines. Responsiveness is big, so i'll focus on mobile design considerations in another post. How big? In some content categories more than half. With User experience i'll focus on A/B testing to ensure and test new and interesting user flows. Page load speed can go hand in hand with accessibility because a more accessible website can / should be simpler and load faster for everyone. I'll be using tools like Google's page speed insights. My last focus will be in web application security. I need to read up more on this subject as it is one of my weaker points but preventing attacks and securing user data is essential. Especially considering the sony / yahoo hacks are fresh in consumers minds.

I'll also focusing on my pasta sauces, a roux is great for thickening white sauces but what can I incorporate to thicken my reds? 2017 will be a year of learning.
